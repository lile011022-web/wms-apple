# 2026-07-13

## 已创建箱子自定义日期筛选

- 在出库装箱页“已创建箱子”工具栏增加开始日期、结束日期和清除操作。
- 允许只设置开始日期或只设置结束日期；同时设置时包含起止两天的完整仓库营业日。
- 日期边界按所选仓库的 `timezone` 换算为 API 所需的 `createdFrom` / `createdTo`，不依赖浏览器所在时区。
- 结束日期早于开始日期时页面会提示错误并停止查询；清除后恢复“未完成”默认视图。

## 当前箱子货物删除

- 当前箱子工作区的每条货物在“编辑”旁新增红色“删除”按钮。
- 删除前必须人工确认；请求执行中同时锁定编辑和删除操作，避免重复提交。
- 删除只允许未封箱或返工中箱子。成功后该货物从箱内移除，库存状态恢复为 `IN_STOCK`，清除装箱时间，并重新出现在可装箱库存中。
- 后端继续写入 `OUTBOUND_BOX_ITEM_REMOVE` 审计日志；本次不修改已有 API 合同或数据库结构。

## 使用逻辑

1. 在“已创建箱子”右上角选择日期，列表自动按所选仓库日期重新查询。
2. 需要恢复日常作业列表时点“清除”，页面返回只显示未完成箱子。
3. 在未封箱的当前箱中找到错装货物，点“删除”并确认。
4. 系统完成删除后更新当前箱数量，该货物可以再次选择或扫码装箱。
