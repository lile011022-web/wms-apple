# 2026-07-12

## 商品表格覆盖更新导入

- 商品管理批量导入新增“按 SKU 覆盖更新已有商品”选项，默认仍为只新增的安全模式。
- 勾选后，已有 SKU 会用表格中的商品名称、品牌、型号、型号代码、分类、颜色、容量、IMEI 要求和 UPC 列表覆盖更新；新 SKU 仍会新增。
- 覆盖更新保留商品原有启用/停用状态；空白可选单元格会清空对应字段。
- UPC 只能继续属于同一 SKU；如果表格尝试使用其他 SKU 的 UPC，整批导入都会拒绝，不会部分更新。
- 导入结果分别显示“新增数量”和“覆盖更新数量”，审计日志保留更新前后快照。

### 使用逻辑

1. 在商品管理导出全部数据，修改 `可重新导入` 工作表。
2. 选择修改后的文件，勾选“按 SKU 覆盖更新已有商品”。
3. 点击“导入并覆盖更新”，成功后页面会显示新增和更新数量。
