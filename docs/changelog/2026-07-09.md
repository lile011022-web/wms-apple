### Usage Logic

1. 默认启动系统时，不会出现“包裹预报”和“包裹预警”菜单。
2. 入库扫码页面不会根据预报单号自动带出客户，操作员按原流程手动选择客户并锁定。
3. 确认入库只执行库存入库、异常和审计逻辑，不再改动预报记录。
4. 如果后续决定继续做包裹预报，在 API 和 Web 环境中同时打开两个开关后重启服务即可恢复入口与联动。
